# Contradiction Detection Protocol

Full sub-process for Step 6 of `SKILL.md`. Consult this before concluding
that two documents contradict each other.

A contradiction should be found only where the factual propositions are
genuinely incompatible — not merely because two documents use different
language or describe different versions of events.

## 1. Identify the factual proposition being tested

State the specific fact or event under review. Do not compare documents
generally — compare specific factual claims.

## 2. Extract the relevant factual claims

Break each document into simple factual propositions. For each, record:

- who acted or spoke;
- what happened;
- when it happened;
- what object, transaction, or event it concerned;
- whether the statement describes a fact, belief, prediction, plan,
  allegation, or recollection.

## 3. Check that the claims are truly comparable

Before finding a contradiction, confirm both statements concern:

- the same person or entity;
- the same event or subject;
- the same time period;
- the same scope or definition;
- the same type of assertion.

If they don't, they aren't comparable — stop here, this isn't a
contradiction.

## 4. Apply the core contradiction test

Ask: **can both statements be true at the same time, under the same
interpretation of the facts?**

- If no → classify as a direct contradiction.
- If both could still be true → do not classify as a contradiction.

## 5. Classify the relationship

Use one of the following labels:

- Direct contradiction
- Tension
- Qualification
- Temporal change
- Difference in knowledge or perspective
- Corroboration
- Unclear or insufficient information

### 5.1 Check the surrounding context

Review the full email thread, attachments, document versions, authorship
and attribution, metadata and dates, and whether the statement was later
corrected, updated, or superseded.

### 5.2 Try to explain the apparent contradiction

Before confirming a contradiction, test plausible alternative
explanations, including:

- the statements refer to different times;
- different people had different knowledge;
- one statement concerns a plan while another concerns actual events;
- one document is a draft, scenario, forecast, or conditional statement;
- the same word is being used differently.

### 5.3 Look for independent evidence

Search for additional documents that help determine which interpretation
is better supported.
